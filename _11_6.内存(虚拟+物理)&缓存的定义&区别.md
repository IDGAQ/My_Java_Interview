# 内存(RAM)&缓存(CachedMemory)
---

```
Q: 你知道缓存、(虚拟+物理)内存的区别么?

A: 
    知道，但在这之前，我们需要知道Windows_CPU的查找顺序：
       一级缓存 -> 二级缓存 -> 其他缓存 -> 内存
  
内存：
    虚拟内存: 硬盘空间上的虚拟内存(存在硬盘上的'冷宫'，离CPU较远，访问频率不高的数据的地方)
    物理内存: RAM(内存条)
    
缓存: 
    L1 Cache: (高速内存，存在CPU里的顶级'正宫'，离CPU最近，与CPU同频，访问最频繁的数据的地方)
    L2 Cache: (存在缓冲处理器BufferStorage中的二级'正宫'，速度比L1慢)
    L3, L4..: (低级'正宫',低速)
    
```
