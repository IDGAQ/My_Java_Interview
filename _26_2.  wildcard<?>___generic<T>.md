# When use wildcard```<?>``` or generic```<T>```

1. use ```<T>``` ,if parameter and return type are related
2. otherwise, use ```<?>```

---

```java
//使用通配符
public static void test(List<?> list) {

}

//使用泛型方法
public <T> void  test2(List<T> t) {
        
}
```
