# int和Integer的区别
---
## Integer和Integer比较时：
```java
public static void main(String[] args) {
        // 新地址，新 1
        Integer a = new Integer(1);
        Integer c = 1;
        
        // b 指向 缓存中 1 的地址
        Integer b = Integer.valueOf(1);
        

        System.out.println(a==b);
        //false
    }
```
Integer比较的是指针，String一样
当Integer<-128或者>127时，也会出现不相等的情况
