# equals(),HashCode()的关系
---

## HashCode的定义和用途：
定义：可以把HashCode想象成object ID number，but it might not be unique<\br>
用途：HashMap, HashSet会通过Object的HashCode进行排列以及搜找

## 区别：
### 1.两个数字的HashCode相等，两个数字不一定相等
### 2.两个数字的HashCode不相等，两个数字一定不相等
伪代码：
···java
if(A.equals(B)){
   A.HashCode(B);
}
if(A.HashCode(B)）{
   A 不一定equals(B)
}
```
