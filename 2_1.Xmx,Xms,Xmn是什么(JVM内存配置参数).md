## Xmx,Xms,Xmn(JVM内存配置参数) ##
---
```
Q:
  -Xmx10240m -Xms10240m -Xmn5120m -XXSurvivorRatio=3
  
  什么意思
```

```
A:
  -Xmx：最大堆大小
  -Xms：初始堆大小
  -Xmn:年轻代大小 / Young Generation(包含Eden/Survivor)

-XXSurvivorRatio：年轻代中Eden区与Survivor区的大小比值

例子中:
  年轻代Xmn大小为5120m, 
  XX - Survivor = 1 : 3
  Survivor区有两块(默认）
  1024 = 5120 / (3x + x + x)
  

```
